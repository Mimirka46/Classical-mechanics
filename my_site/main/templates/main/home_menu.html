Обучающий видеоролик на тему "Кинематика"
Равнопеременное прямолинейное движение

1. a = 𝒗 – 𝒗₀ / t - ускорение

2. x = ±x₀ ± 𝒗₀t ± at/2 - уравнение Галлилео

3. 𝒗 = ±𝒗₀ ± at - уравнение скорости

4. 2Sa=𝒗²-𝒗₀²

5. 𝒗ср = S₁ + S₂ + S₃ … / t₁ + t₂ + t₃ … - средняя скорость

a - ускорение (м/с²)

𝒗 - скорость (м/с)

t - время (c)

S - путь (м)

Движение по окружности:

1. a = 𝒗 – 𝒗₀ / t - ускорение

2. x = ±x₀ ± 𝒗₀t ± at/2 - уравнение движения

3. 𝒗 = ±𝒗₀ ± at - уравнение скорости

4. 𝒗ср = S₁ + S₂ + S₃ … / t₁ + t₂ + t₃ … - средняя скорость

5. S = x − x₀ = 𝒗t - перемещение

Движение по окружности:

1. w= l / t - Угловая скорость

2. 𝒗=wR - скорость

3. a = 𝒗² / R - центростремительное ускорение

Обучающий видеоролик на тему "Динамика"
Динамика

1. ma = F₁ + F₂ + ... + Fn - Второй закон Ньютона

2. m₁a₁=-m₂a₂ - Третий закон Ньютон

3. F = kx - Закон Гука

4. Fтр = ±𝒗₀ ± at - Модуль силы трения

5. F = G (m₁m₂)/R² - Закон всемирного тяготения

F - сила (Н)

R - радиус планеты (Н)

m - масса (кг)

k - коэфициент упругости

x - длина растяжимости (м)

G - гравитационное постоянство (Н*м²/кг²)

G = 6,67/10¹¹ (Н*м²/кг²)

Обучающий видеоролик на тему "Статика"
Статика

1. M=Fl - Момент силы

2. M₁+M₂+M₃+...=0 - Правило моментов

3. F₁+F₂+F₃+...=0 - Первое условие равновесия

M-момент силы (Н*м)

l-плечо силы (м)
